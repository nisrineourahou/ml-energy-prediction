# AI-Driven Predictive Maintenance & Energy Efficiency Software

This repository contains the core components of an AI-driven predictive maintenance software solution designed to optimize operational efficiency and track real-time energy metrics in commercial properties.

## Project Structure & Files
* **`finalpredectionmldl.py`**: The main Python script featuring the Machine Learning architecture, UI integration, and data diagnostic logic.
* **`201608.csv`**: The dataset containing historical operational and energy efficiency metrics utilized for model training.
* **`rapport_analyse_energetique.txt`**: Detailed energy analysis documentation and system performance reporting.

## Key Features
* **Machine Learning Integration**: Built predictive modeling scripts to anticipate infrastructure failures and lower utility overhead.
* **Interactive UI Indicators**: Processed real-time data inputs to supply stakeholders with diagnostic insights.
* **Data Automation**: Cleaned and handled time-series CSV metrics for consistent diagnostic execution.

## Tech Stack
* **Language**: Python
* **Concepts**: Predictive Maintenance, Machine Learning, UI/UX Engineering, Energy Analytics
